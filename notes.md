# Anaconda

Anaconda can create virtual envs for us with diff dependencies and python versions.

- activate a conda env
  - `conda acvtivate ENV`

- list conda envs
  - `conda env list`

# Jupyter Notebooks

- open a notebook
  - `jupyter notebook`
    - opens `http://localhost:8888/tree` for the tree you are in
